# Objectives

This project aims to build a practical Azure Administration portfolio demonstrating the ability to:

- Deploy and manage Azure resources
- Organize environments using Resource Groups and naming conventions
- Implement Azure governance and cost management
- Configure Azure RBAC following the Principle of Least Privilege
- Deploy and manage Azure Storage services
- Create and administer Azure Virtual Machines
- Configure Azure networking components
- Implement Azure monitoring and diagnostics
- Automate Azure deployments using Azure CLI, Azure PowerShell, ARM Templates and Bicep
- Document validation, troubleshooting and cleanup procedures
